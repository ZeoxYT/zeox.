FAQ

Frequently Asked Questions

GeneralSecurityPricingCommunity
Do I need to know how to code to use this?

No. Upload your .lua file, click encrypt, and copy your raw link. No coding knowledge is required.

What exactly does encryption protect me from?

It makes your source unreadable to anyone who tries to open it directly, so your logic and variable names aren't exposed.

What is a raw link and how does it work?

It's a permanent URL that serves your encrypted script. You paste it into your loader just like any normal raw file link.

Will my encrypted script still run in my loader?

Yes. Encrypted scripts run exactly the same as the original — only the readability of the source changes.

Can grabber tools still access my raw file?

Direct fetch attempts from outside your loader are blocked automatically, keeping casual grabbing tools from pulling the file.

Is my source code stored securely?

Yes. Files are stored encrypted at rest and only ever served through the protected raw link system.
